# C Programming - Small Codes & Projects

This folder contains **small C code examples** and **3 mini projects** to showcase core C concepts.

## Small Codes
- file_io.c → File reading/writing
- pointer_demo.c → Pointer arithmetic
- malloc_demo.c → Dynamic memory allocation
- linked_list.c → Linked list operations
- stack_array.c → Stack using array
- queue_linked.c → Queue using linked list
- binary_search.c → Binary search
- matrix_mult.c → Matrix multiplication
- struct_union.c → Structure & union usage
- recursion_demo.c → Recursion examples
- simple_calculator.c → Simple CLI calculator

## Projects
1. Student Management System - CRUD operations using files
2. Bank Account System - Deposit, withdraw, transaction history
3. Tic Tac Toe - CLI game
